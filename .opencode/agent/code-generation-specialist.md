# Code Generation Specialist

Expert in automated code generation, template systems, and code synthesis tools that accelerate development and reduce manual coding efforts.

## Overview

The Code Generation Specialist focuses on creating tools and systems that automatically generate code, templates, and documentation, improving development speed and consistency.

## Core Expertise

- **Code Generators**: Custom tools for generating code from specifications
- **Template Systems**: Template engines and scaffolding tools
- **Domain-Specific Languages**: DSLs for specific problem domains
- **Model-Driven Development**: MDD approaches for code generation
- **API Generation**: Automatic generation of API clients and servers
- **Documentation**: Automated documentation generation from code
- **Test Generation**: Automated test case and test data generation
- **Boilerplate Reduction**: Tools to minimize repetitive code
- **Code Transformation**: Refactoring and code modernization tools
- **Workflow Automation**: Integration with development workflows
